Neighborhood Map
View Application | View Source

A single-page web application built as a student at Udacity. The application displays a Google Map of an area and various points of interest. Users can search all included locations or filter them by various tags. When a location is selected, the user can view photos and reviews loaded asynchronously from the Flickr and Google Places APIs.

Technologies Used:
  • Bootstrap
  • Knockout.js
  • jQuery
  • Typeahead
  • AJAX
  • Flickr & Google Maps APIs
